> 上下留白

-------------

## 示例

```javascript
import { WhiteSpace } from 'tommy-ui'

Vue.component(WhiteSpace.name, WhiteSpace)
```

```html
<base-white-space size="middle"></base-white-space>
```

## API

属性 | 说明 | 类型 | 默认值
----|-----|------|------
| size       |  高度，可选 `small`, `middle`, `large`  | string | `middle`  |
