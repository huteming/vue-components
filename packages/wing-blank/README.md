# WingBlank

-----------------------

## 例子

```javascript
import { WingBlank } from 'tommy-ui'

Vue.component(WingBlank.name, WingBlank)
```

```html
<base-wing-blank size="middle">内容</base-wing-blank>
```

## API

属性 | 说明 | 类型 | 默认值
----|-----|------|------
| size    | 两翼留白的间距，可选 `small`, `middle`, `large`  | string |  `middle`  |
