> 图标

-------------

## 例子

```javascript
import { Icon } from 'tommy-ui'

Vue.use(Icon)
// Vue.component(Icon.name, Icon)
```

```html
<base-icon icon="success"></base-icon>
```

## API

| 属性 | 说明 | 类型 | 默认值 |
|-----|-----|------|------|
| icon | 图标 | String | |
| size | 大小 | String | `16px` |
| color | 颜色 | String | |
